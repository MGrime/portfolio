Velocity Game Engine (Vulkan/C++)

Project Overview

Source Code - MGrime/Velocity: A Proper Implementation for my Master's Project. (github.com)

Project Goal - To create a game engine capable of creating a standalone simple game, with a particular focus on implementing a Physically Based Rendering (PBR) Pipeline. Also to create a demonstration game using the engine, along with a level editor built on top of the engine.

Concepts Used - Physically Based Rendering, Serialisation, Entity-Component System, Optimisation, Large codebase & Project management, Asset importing

Project Technologies

Tools - GitHub / Visual Studio / RenderDoc / Nvidia NSight Graphics / Premake5

Languages - C++ / GLSL / Lua

Libraries - VulkanHPP / glm / Dear ImGui / Snappy / GLFW

Time Allowed - 11 Months
