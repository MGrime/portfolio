Game In A Day (UE4 / C++)

Project Overview

Source Code - Lost in system corruption.

Project Goal - To implement a fully completed mini-game within a single day to test and demonstrate my progress in learning Unreal Engine and C++.

Concepts Used - Time management, Asset creation, Real-time spawning of actors, Rapid prototyping

Project Technologies

Tools - GitHub / Git LFS / Visual Studio / Unreal Engine 4 / Audacity

Languages - C++

Time Allowed - 10 hours

Discussion

This was a fun project. As part of the Advanced Game Development module we were challenged to create a complete minigame using what we have learned this year in Unreal Engine 4 in only a single day. For this concept I chose to make a simple aim training game.

I decided that a 10 hour limit would be a good balance. I split the time into 8 hours of implementation of the game mechanics and 2 hours of polish of the sound and art.

I recorded many of the sound effects myself, primarily the gun firing sound, the beeping countdown clock and the pop sound effect of the targets. To do this I used Audacity to record the sound and do some simple sound editing. This was my first time creating my own assets and I am happy with the results.

For such a time-limited project I am very proud of what I managed to implement. A video of the completed game can be seen below.
