Heroes Of Joro (Unity/C#)

Project Overview

Source Code - MGrime/Heroes Of Joro: Unity 3D Dungeon Crawler with procedural generation and a dynamic party system.. (github.com)

Project Goal - To create a dungeon crawler game with a party system and procedurally generated dungeons as part of a team.

Concepts Used - Procedural generation, Gameplay mechanics, Navmesh, Large asset management, Optimisation, Unity Physics, Collision resolution

Project Technologies

Tools - GitHub / Visual Studio / Unity 2019

Languages - C#

Time Allowed - 9 months (part-time)

Discussion

Development is complete as of May 5th 2021, which met our original deadline for this project.

My primary contribution in this project was the development of the dungeon generation algorithm, which is a custom algorithm specifically to fit our games’ needs. It is capable of generating an unlimited number of interconnected floors using a set list of room “prefabs”. The algorithm makes use of Unity’s physics system and collision volumes to detect and manage overlap, allowing the creation of large sprawling dungeons with no overlap between the placement of the rooms.
