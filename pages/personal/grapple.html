Grapple (Unreal Engine 4/C++)

Project Overview

Source Code - MGrime/Grapple Grapple is a simple 6-level free running game made in UE4 and C++ (github.com)

Project Goal - To implement a more polished and extensive game in Unreal Engine 4/C++ to show current knowledge and skills with these tools.

Concepts Used - Gameplay Design, Project Management, Level Design, Serialisation, Behaviour Trees

Project Technologies

Tools - GitHub / Git LFS / Visual Studio / Unreal Engine 4 / Audacity

Languages - C++

Libraries - Eigen, TL-Engine, SDL (Subsystems)

Time Allowed - 1.5 months (Complete)

Discussion

Grapple is a simple 6-level free running game, implemented in Unreal Engine 4 and C++.

The game has been designed with a very tight scope so it can be released as a small but highly polished product. A lot of time has been spent creating a satisfying and feature rich player controller.

The player has fully animated 360 degree movement, with the ability to crouch and double jump. All animations are handled using a combination of Unreal Engine's blend spaces, and the animation blueprint event graph to blend using state extracted from the C++ code.

Another key aspect of the player controller is the ability to wall run on any surface of the correct angle. This is the crux of the gameplay, and thus there has been a strong focus on making the system intuitive to understand and satisfying to play.

The final game contains 6 levels, with tokens to collect in each one, a level select screen that displays your best results for time and tokens collected, and enemy AI that was created using Unreal Engine's Behaviour Trees.
