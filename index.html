Tetra

Tetra is a spirtual successor to my Velocity Game Engine from my master's project. It follows a very similar style, but with the intent to develop much further. Here are some planned features:

- Cross-Platform support with multiple backends (DX11/DX12/OGL/Vulkan/Metal)

- Custom ECS built for performance and a heavily scoped feature set

- Support for bypassing the editor and writing fast game code in C++ using the core engine

- Lua Scripting support
